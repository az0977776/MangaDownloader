MangaDownloader
======

WIP

downloads manga from mangareader.net and saves as JPG and pdf

### Installation
Python 3.6

Installing the project itself:
```sh
git clone https://github.com/az0977776/MangaDownloader.git
```


To install dependencies:
```sh
pip install -r requirements.txt
```

Running the program:
```sh
python main.py
```
